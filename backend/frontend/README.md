# Frontend – Sistema de Locação de Caçambas

Este diretório contém o frontend do sistema de locação de caçambas.

## Objetivo do Frontend
Fornecer uma interface simples e intuitiva para o cadastro de locações de caçambas.

## Funcionalidades Previstas
- Tela de cadastro de locação
- Formulário para preenchimento dos dados do cliente
- Envio das informações para o backend
- Exibição de mensagens de sucesso ou erro

## Tecnologias
- HTML
- CSS
- JavaScript

## Status
Frontend em fase inicial de desenvolvimento.
